# Character Image Upload Instructions

## Image Specifications

| Property        | Requirement                                    |
|-----------------|------------------------------------------------|
| **Dimensions**  | 1080 x 1920 px (9:16 vertical / portrait)      |
| **Format**      | PNG (preferred) or WebP                        |
| **Background**  | Transparent or dark (#050505) — no white       |
| **Style**       | Realistic POV portrait of an AI girl           |
| **Framing**     | Upper body / bust, slight tilt, soft lighting  |
| **Mood**        | Sensual, warm, intimate — soft gaze at camera  |
| **File size**   | Under 500 KB (optimized for web)               |

## Upload Path

Place the image at:

```
apps/web/public/store/vip-character.png
```

The subscription page will automatically pick it up once uploaded.

## Placeholder

Until the image is uploaded, a dark gradient placeholder is rendered in its place.
